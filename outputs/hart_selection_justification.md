# Chart Selection Justification

## 1. Sales Trend

**Chart Used:** Line Chart

**Business Question**

How are sales changing over time?

**Why Appropriate**

Line charts clearly display trends and seasonality.

**Fields Used**

* X-axis: Order Date
* Y-axis: Sales
* Color: Region

**Design Principle**

Emphasizes trends with minimal visual clutter.

**Mistake Avoided**

Did not use pie charts for time-series data.

---

## 2. Regional Performance

**Chart Used:** Horizontal Bar Chart

**Business Question**

Which regions contribute the highest sales and profit?

**Fields Used**

* Region
* Profit
* Sales Labels

**Design Principle**

Sorted bars improve comparison.

**Mistake Avoided**

Avoided unsorted categorical charts.

---

## 3. Category Profitability

**Chart Used:** Horizontal Bar Chart

**Business Question**

Which product categories generate the highest profit?

**Fields Used**

* Sub Category
* Profit
* Category Color

**Design Principle**

Easy comparison between multiple categories.

**Mistake Avoided**

Avoided 3D charts and unnecessary visual effects.

---

## 4. Customer Segment

**Chart Used:** Bar Chart

**Business Question**

Which customer segment contributes most to sales?

**Fields Used**

* Customer Segment
* Sales

**Design Principle**

Simple comparison of categorical values.

**Mistake Avoided**

Avoided complex stacked visuals.

---

## 5. Shipping Performance

**Chart Used:** Bar Chart

**Business Question**

Which shipping mode experiences longer delivery times?

**Fields Used**

* Ship Mode
* Average Delivery Days

**Design Principle**

Highlights operational efficiency.

**Mistake Avoided**

Avoided using averages without clear labels.

---

## 6. Discount vs Profit

**Chart Used:** Scatter Plot

**Business Question**

How does discount affect profitability?

**Fields Used**

* Discount
* Profit
* Sales (Size)
* Category (Color)

**Design Principle**

Shows relationship and outliers effectively.

**Mistake Avoided**

Avoided misleading line charts for correlation analysis.

---

## 7. Return Analysis

**Chart Used:** Highlight Table / Bar Chart

**Business Question**

Which products or categories have higher return rates?

**Fields Used**

* Category
* Return Rate

**Design Principle**

Uses color intensity to highlight problem areas.

**Mistake Avoided**

Avoided displaying raw counts without context.

---

# Dashboard Design Principles Applied

* Consistent colour palette
* Proper alignment
* Clear KPI hierarchy
* Interactive filters
* Dashboard actions
* Appropriate chart selection
* Business-focused storytelling
* Readable labels
* Minimal clutter
* Executive-friendly layout

These visualization choices ensure that leadership can quickly identify trends, risks, and opportunities while making data-driven decisions.
